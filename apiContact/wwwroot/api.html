Route Groups

40+ endpoints across 9 groups — click any card in the Playground to try it live or see Docs for full schemas

Auth
POST/api/auth/register

Register a new account. Returns JWT + refresh token pair.

POST/api/auth/login

Login and receive access + refresh token. Field: usernameOrEmail.

POST/api/auth/refresh

Silently renew access token via refresh token rotation.

GET/api/auth/me

Return authenticated caller's full profile.

Users
GET/api/users

List all users sorted by display name.

GET/api/users/online

Currently online users only.

PATCH/api/users/{id}

Partially update display name, bio, or avatar URL.

POST/api/users/{id}/status

Set online / away / offline — broadcasts via WebSocket.

POST/api/users/{id}/block

Block a user — they can no longer send you DMs.

DELETE/api/users/{id}/block

Unblock a previously blocked user.

GET/api/users/blocked

List all users you have blocked.

Rooms
GET/api/rooms

All rooms sorted by latest activity.

GET/api/rooms/mine

Rooms the authenticated user has joined.

POST/api/rooms

Create a Group or Channel room with auto-slug generation.

POST/api/rooms/dm

Create or fetch an existing Direct Message room with another user.

POST/api/rooms/{id}/invite

Generate a shareable invite code with optional expiry.

POST/api/rooms/join

Join a room via invite code.

GET/api/rooms/{id}/stats

Room stats: member count, message count, active users.

DELETE/api/rooms/{id}/members/{userId}

Remove a member from a room (owner / admin only).

Messages
GET/api/messages/room/{roomId}

Paginated message history — oldest to newest.

POST/api/messages

Send a message; auto-broadcasts via SignalR to all room members.

PATCH/api/messages/{id}

Edit a message — broadcasts MessageEdited event.

DELETE/api/messages/{id}

Soft-delete — content replaced with "[Message deleted]", history preserved.

POST/api/messages/{id}/pin

Pin a message — stored with pinned-at timestamp and pinner info.

DELETE/api/messages/{id}/pin

Unpin a previously pinned message.

GET/api/messages/room/{roomId}/pinned

List all pinned messages in a room.

GET/api/messages/{id}/replies

Fetch replies threaded to a parent message.

Files
POST/api/files/upload

Upload a file (multipart/form-data, max 20 MB). Returns public URL. Sandboxed storage.

DELETE/api/files/{fileName}

Delete a file — path-traversal safe, rate limited per user.

Audit
GET/api/audit/recent

Latest N audit events. Admin only. Query: ?limit=N.

GET/api/audit/search

Filter by user, action type, and date range. Admin only.

Health & Meta
GET/health

Public health check — returns 200 OK when the API is up.

GET/swagger

Interactive Swagger UI with full OpenAPI spec.

WebSocket Hub
WS/hubs/chat?access_token=…

SignalR hub. Pass JWT via access_token query param. Supports automatic reconnect.

ONReceiveMessage · MessageEdited · MessageDeleted

Incoming message events — fired for all members of the room.

ONUserTyping · UserStoppedTyping · UserOnline · UserOffline

Presence and typing indicator events broadcast to room members.

INVOKEJoinRoom · LeaveRoom · Typing · StopTyping · Ping

Client-to-server hub methods. Call via conn.invoke("JoinRoom", roomId).

Base URL: All REST endpoints are at [origin]/api/*. The WebSocket hub is at [origin]/hubs/chat. All protected routes require Authorization: Bearer <token>.
