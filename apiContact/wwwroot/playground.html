Quick Tips

Start with Login

Hit the Login tab first. The JWT is captured automatically and injected into every subsequent authenticated request — no copy-paste needed.

Keyboard Shortcut

Press Ctrl+Enter while editing the request body to run the request instantly.

Copy Response

Click Copy in the response panel to grab the full JSON to your clipboard for use in other tools or curl commands.
